## ğŸš€ Installation & Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/netz-sg/pantry.git
   cd pantry
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   ```

   ```bash
   npm run dev
   ```
